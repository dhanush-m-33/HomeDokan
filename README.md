# HomeDokan
HOMEDOKAN is a responsive grocery e-commerce website built with HTML, Bootstrap 5, and JavaScript, featuring product search, shopping cart, Local Storage-based user authentication, reusable components, and a modern responsive UI.

HOMEDOKAN – Grocery E-Commerce Website

HOMEDOKAN is a responsive front-end grocery e-commerce website built using HTML, Bootstrap 5, and JavaScript. The project provides a modern shopping experience with product browsing, search functionality, shopping cart management, user authentication using Local Storage, and a responsive design suitable for desktops, tablets, and mobile devices.

Features
- Responsive Home, About, Products, Contact, Login, Signup, Cart, Checkout, and My Account pages
- Product search with auto-suggestions
- Smooth scrolling to searched product with highlight animation
- Hopping cart with Add to Cart, quantity increase/decrease, and automatic total calculation
- Cart data stored using Local Storage
- User Login, Signup, Logout, and My Account using Local Storage
- Dynamic cart badge showing total items
- Reusable global Navbar and Footer using HTML injection
- Fully responsive layout built with Bootstrap 5
- Beginner-friendly and modular JavaScript code

Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Local Storage API
